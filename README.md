# pbsmanage
## usage: pbsmanage.py [-h] -d DIRET [-p PROCESS]
## To Manage pbs scripts.
# optional arguments:
```
-h, --help            show this help message and exit
-d DIRET, --diret DIRET
                        The directory name where the pbsfiles save in
-p PROCESS, --process PROCESS
                        The process number.
```
